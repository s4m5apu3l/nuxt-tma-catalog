# Product Overview

## TMA Telegram Catalog

A Telegram Mini App (TMA) catalog system for product/service rentals with administrative panel.

### Core Purpose

- **User-facing catalog**: Browse categories and products through Telegram Mini App
- **Admin panel**: CRUD operations for categories and products
- **Telegram integration**: Direct contact with admin through bot
- **Multi-language support**: English and Russian localization

### Key Features

- Product catalog with categories and detailed product pages
- Image galleries with multiple photos per product
- Price display with rental units (hour/day/week/month)
- Availability status tracking
- Admin authentication via Telegram ID
- Responsive design optimized for mobile (Telegram WebView)

### Target Users

- **End users**: Browse catalog through Telegram Mini App
- **Administrators**: Manage content through web admin panel
- **Business owners**: Rental service providers using Telegram as primary communication channel

### Business Model

Rental marketplace where users browse products and contact admin directly through Telegram for booking/inquiries.
